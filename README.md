# PRODIGY_AD_04 - Tic Tac Toe Game
Tic Tac Toe game built with Kotlin for Prodigy InfoTech Android Development Internship Task 04.

## Features
- Two players: X and O
- 3x3 grid
- Win detection (row, column, diagonal)
- Draw detection
- Reset game

## Tech Stack
- Kotlin
- XML
- GridLayout
